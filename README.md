# 👋 Introduce

> 독서를 즐기고 독후감을 공유하며 다양한 경험을 나눌 수 있는 독서 커뮤니티 플랫폼을 소개합니다.<br/>
> 이 플랫폼은 사용자들이 자신의 독서 활동을 기록하고 다른 이용자들과 공유하는 공간으로 디자인되어 있습니다.

**🗓️ 작업기간**

`2023/11/21` ~ `2023/12/17`<br/>

**⚙️개발 환경 세팅**

> $ npm install <br/>
> $ npm run dev
